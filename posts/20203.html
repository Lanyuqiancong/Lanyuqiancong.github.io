github博客备份问题
发表于2020-08-15|更新于2020-10-07|教程hexo
|阅读量:

如果换新电脑了一定！一定！一定！要备份！！！这里我大致摸索了两种方法~

首先要安装必要的软件

  • Git客户端
    https://git-scm.com/
  • Node JS
    https://nodejs.org/en/

在github官网添加新电脑产生的密钥，并测试是否能连接

  • 配置git个人信息并生成ssh密钥

在Github设置ssh密钥

密钥在.ssh文件夹下的id_rsa.pub里面
打开GitHub网页，在右上角Seetings - SSH and GPG keys - NEW SSH key，把复制的内容粘贴到key栏，然后保存即可

将原来电脑上个人博客目录的以下内容拷到新电脑上，注意无需拷全部，只拷以下几个目录

在Git Bash安装hexo

  • npm install hexo-cli -g

在新博客目录下安装相关模块

最后运行hexo clean & hexo g & hexo d 看看是否成功

补充说明

写个人博客一定要有常备份的习惯，说不定哪天你电脑突然就坏了呢（滑稽），这个时候你的这种习惯就会拯救你，下面介绍我自己摸索的方法，以butterfly主题为例~
